Interactive MBTA Map : A map with the MBTA red line overlaid. Users can click on red line stops to display real-time train schedule data, and click on their own pin to view the nearest red line stop. Make sure you click "allow" if your browser asks you to allow the access of your location. If the map doesn't load, make sure you're not using chrome - some extensions conflict with this application. You can view the code on my GitHub.
